## Installing

1. Download Go from golang.org to usr/local folder

2. Unzip Go file

```bash
tar -C /usr/local -xzf nome.tar.gz
```

3. Export /usr/local/go/bin` to path

```bash
vim ~/.profile
export PATH=$PATH:/usr/local/go/bin
```

4. Refresh Path

```bash
source ~/.profile
```

5. Testing

```bash
go version
go env | grep GOPATH
```
